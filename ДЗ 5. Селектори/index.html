Cелектори

CSS-селектори використовуються для стилізації HTML елементів на веб-сторінці. Існує широкий вибір CSS-селекторів, що дозволяє максимально точно підібрати елементи для стилізації.

У нас є як мінімум 5 типів селекторів:

  1. Універсальні селектори CSS
  2. CSS селектори за назвою тегу
  3. CSS селектори за класом
  4. CSS селектори за атрибутами
  5. CSS селектори по id

1.Універсальні селектори CSS позначаються звичайною зірочкою "*". Під зірочкою може матися на увазі будь-який елемент розмітки HTML-сторінки. Ми звертаємось до всіх елементів на сторінці. Має найнижчій пріоритет. Наприклад: * {color: red;}

2.CSS селектори за назвою тегу. Звертаємось до певного тегу. Має вищій пріоритет ніж універсальний тег з "*".Запис селекторів за тегами виглядає так: h1{color: blue;} або p {color: green}

3.Наступними у нашому списку видів селекторів CSS йдуть селектори за класами. Нагадаємо, що "class" - це атрибут HTML-тега, він дуже зручний тим, що може застосовуватися до однотипних елементів. Назва класу може бути будь-яким, головне, щоб вона відповідала тому елементу, який описується CSS-правилом. Взагалі атрибут "class" вважається найголовнішим інструментом CSS. Під час створення CSS-правила точка «.» перед ім'ям класу є обов'язковим. Наприклад: .u-list li {color: brown;}

4.Далі будуть розглянуті CSS селектори за атрибутами HTML-тегів. Селектори по атрибутам рівносильні з селекторами по класу. Їх запис становить: p [data-text-type] {color: black;}

5.CSS селектори по id. Цей атрибут застосовується до індивідуального елементу, тобто. елемент з id може бути в єдиному екземплярі на сторінці згідно з правилами семантики. У каскадних таблицях стилів ці атрибути використовуються вкрай рідко. Cелектор по id у CSS-правилі оформляється так: #heading-2 {color: yellow;}
